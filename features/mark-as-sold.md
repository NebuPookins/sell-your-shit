# Behavior

"Mark as Sold" should be on the item level, not the listing level. For example,
if I mark an item as sold on Kijiji, that means the item is sold, and so the
listing on Kijiji, Craigslist, Facebook, etc. all should no longer be active.

The Dashboard can list all fo these listings as needing action: The user can
then go through all the listings and update them to mark the item as sold.

If an item has been sold, then we should no longer show that the corresponding
listings need renewal, or that a price reduction is due on the listing.
